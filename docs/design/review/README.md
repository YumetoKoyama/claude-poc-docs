## 読み方

1. **全体を把握したい** → `00-設計サマリー.md` のアーキテクチャ図・主要業務フローから
2. **特定の画面をレビューしたい** → `01-画面一覧と遷移.md` で画面 ID を確認 → `screens/SCR-XXX-*.md`
3. **API の仕様を確認したい** → `02-API仕様サマリー.md`（詳細な入出力スキーマは元 YAML を参照）
4. **DB 構造を確認したい** → `03-データモデルサマリー.md`

このディレクトリの内容に誤り・不足を発見した場合、**このディレクトリのファイルを直接編集しないでください**。
