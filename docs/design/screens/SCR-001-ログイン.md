# 画面設計: SCR-001 ログイン画面

> ID 凡例: [docs/凡例.md](../../凡例.md) 参照

## 画面概要

- 画面 ID: SCR-001
- URL パス: `/login`
- 対応ユースケース: UC-004（ログイン）, UC-005（ログアウト、遷移先として）
- 関連機能要件: `docs/requirements/functional/認証.md`
- 所属レイアウト: 認証前独立レイアウト

## 入力

| 項目名 | 型 | 必須 | 送信先（operationId.フィールド） | 説明 |
|-------|----|------|--------------------------|------|
| ログイン ID | 文字列 | 必須 | `login.loginId` | |
| パスワード | 文字列 | 必須 | `login.password` | |

## 出力・表示内容

| 項目名 | 型 | データ源（operationId.フィールド） | 説明 |
|-------|----|--------------------------|------|
| エラーメッセージ | 文字列 | `login` の 401 `ErrorResponse.message`（MSG-015, MSG-016） | 認証失敗・ロック中の表示 |

## バリデーションメッセージ

| 項目名 | 条件 | メッセージ |
|-------|------|-----------|
| ログイン ID / パスワード | 認証失敗 | MSG-015: ログイン ID またはパスワードが正しくありません。 |
| ログイン ID / パスワード | 連続5回失敗によるロック中（Q-NF2） | MSG-016: ログイン試行回数が上限に達したため、しばらくログインできません。 |

## 画面遷移（この画面から）

| トリガー | 遷移先（画面 ID） | 条件 |
|---------|--------------------|------|
| ログイン成功（配送依頼企業） | SCR-006 | `login.user.tenantType == REQUESTER` |
| ログイン成功（運送会社） | SCR-013 | `login.user.tenantType == CARRIER` |
| 「新規登録」リンク | SCR-002 | — |
| 「パスワードをお忘れですか」リンク | SCR-003 | — |
| ログアウト操作（共通ヘッダー由来） | SCR-001 | 任意の認証済み画面からログアウト成功時 |

## API 対応

| operationId | HTTP メソッド | パス | 概要 |
|-------------|-------------|------|------|
| login | POST | /api/auth/login | ログイン認証・トークン発行 |
| logout | POST | /api/auth/logout | ログアウト（共通ヘッダーから呼び出し） |

## セキュリティ考慮事項

- 認証: 不要（本画面自体は未認証で到達）
- 認可: なし
- CSRF 対策: JWT Bearer トークン方式のため Cookie を用いず、CSRF の主要リスクは低減される（セキュリティ設計.md 参照）
- その他: パスワード入力欄は `type=password` かつ `x-sensitive: true`（API 側 writeOnly）。ログイン失敗はユーザー列挙防止のため「ID またはパスワードが誤り」という一律メッセージとする（存在しない ID との差異を出さない）

## 受け入れ条件

| AC-ID | 区分 | Given（前提状態） | When（操作） | Then（期待結果） | 関連 BR | operationId |
|-------|------|-----------------|------------|----------------|--------|-------------|
| AC-001 | 正常系 | 発行済みの有効なログイン ID・パスワードを持つ状態 | ログイン操作を行う | 認証に成功し自テナントのダッシュボードへ遷移する | — | login |
| AC-002 | 正常系 | ログイン中の状態 | ログアウト操作を行う | セッションが終了しログイン画面へ遷移する | — | logout |
| AC-101 | 異常系 | ログイン ID またはパスワードが誤っている状態 | ログイン操作を行う | MSG-015 が表示されログインできない | — | login |
| AC-201 | 境界値 | ログイン失敗が連続5回に達した状態（Q-NF2） | さらにログインを試行する | 簡易ロック機構により15分間ログインを拒否しMSG-016を表示する | — | login |
